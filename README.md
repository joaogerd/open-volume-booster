# Open Volume Booster

Projeto Android simples, limpo e sem anuncios, inspirado em aplicativos de controle de volume e reforco de audio.

## Objetivo

Criar uma primeira versao pequena, facil de manter e com interface direta.

## Funcionalidades iniciais

- controle de volume;
- controle de reforco;
- interface simples;
- codigo aberto;
- sem banners de propaganda.

## Estado atual

Estrutura inicial do projeto Android criada.
